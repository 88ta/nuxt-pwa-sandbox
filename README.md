https://nuxt-pwa-sandbox.appspot.com/


## Feature
SPA
SSR
PWA
Serverless(GAE/SE)

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
